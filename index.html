№ Типы Селекторов Описание CSS Пример
1 По тегу Селектор по тегу находит элемент на странице по имени тега. 
span{
background-color: blue;
    }

Это первый абзац текста.

А это уже второй.
2 По классу Селектор по классу находит элемент на странице по значению атрибута class. Самое важное — поставить точку перед значением атрибута class того элемента, к которому вы хотите применить стили. 
.class{
 color: red;
      }

3 По идентификатору Селектор по идентификатору находит на странице элемент, которому задан атрибут id с конкретным значением. 
 #first {
 color: yellow;
        }

Эстетика - наука о законах красоты.

Только сяду за учёбу — обязательно разбудят.
4 Потомка Цепочка перечисленных через пробел селекторов обозначает вложенность от родительских элементов к потомку. Это позволяет управлять стилями вложенных элементов. HTML элемент, расположенный внутри другого элемента является его потомком. Элемент <title> потомок <head>, а элемент <h2> и <p> потомки <body>. Элементы пишутся через (" ") пробел. 
p a {
    color: violet;
    }

5 Универсальный Селектор * (универсальный селектор) позволяет выбрать все элементы внутри документа. Селектор также может выбрать все элементы внутри другого элемента. 
*{
    color: green;
 }
6 По атрибуту Селектор по атрибуту находит элемент на странице по значению либо по наличию атрибута. 
blockquote[cite]{
    background-color: gray;
    color: gold;
                 }

7 Комбинированные Можно задать единый стиль не только одному селектору, но и группе селекторов. При перечислении селекторов необходимо обязательно разделять их запятой. 
.selector,
code,
#id,
blockquote[cite]{
  color: red;
}
